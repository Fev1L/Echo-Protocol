# Echo Protocol

**Echo Protocol** is a 2D post-apocalyptic horror game developed in **C++ using SDL3**.

The project was created as a personal learning project while studying the fundamentals of game development and low-level programming without using a game engine.

---

## Game Concept

The game takes place in a **post-apocalyptic world controlled by hostile AI systems**.  
Most of humanity has disappeared or gone into hiding.

You play as a survivor who has taken refuge in an **old Soviet bunker**.

Your objective is simple:

**Survive for 7 nights until a rescue group arrives.**

The bunker is equipped with a monitoring system that periodically emits an **echo signal**.  
This signal scans the surrounding area and reveals enemy positions.

Using this information, the player must **deploy decoys to distract enemies and keep them away from the bunker**.

Survival depends on observation, timing, and strategic use of limited resources.

## Learning Goals

The project was developed as a **final project for a personal course on C++ game development**.

The main goal was to **deeply understand how games work internally without relying on game engines**.

During development I focused on:

- understanding the **game loop architecture**
- working with **low-level rendering**
- building gameplay systems directly in **C++**
- implementing basic **AI behavior**
- structuring a small game project from scratch

The project significantly improved my understanding of **C++ programming and game development fundamentals**.
